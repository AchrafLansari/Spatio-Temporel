Plan
Satellite

Click a photo on a map to see it in detail. To navigate around the map, you can:
  • Double click to zoom in
  • Drag the map to pan
  • Click the control on the left and drag a rectangle to zoom into a particular region
  • Pop out the search control on the bottom and enter a city name or address

Acknowledgements

  • Google Maps Photo Layer example. This is just a modification and addition of some components to such example.
  • SIMILE Widgets: Timeline component. The Timemap component (to ease the combination of Timeline + Google Maps) is not being used at the moment.
  • Datejs: A javascript library to operate with Date data.
  • Flickr: For their API and the ability to filter / query using so many parameters. The most important in this case, the date when the photo was taken.

Comments

  • Note to MSIE users: The author has limited access to MS products, so this site has not been tested with such a particular bowser. If you experience any problem, we suggest you to use another browser. We do recommend Firefox.
  • The next goal is to include images from additional sources. Other photo repositories allow searching based on geographic coordinates. These could become excelent repositories of images for this application if they sometime allow to search photos based on the date of shooting.
